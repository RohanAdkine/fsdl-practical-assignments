# ChronoX - Premium Smartwatches Ecommerce

<div class="header">
  <p align="left">
    A modern, responsive, and beautifully designed landing page for a premium smartwatch ecommerce store. Built with core web technologies, it features dynamic product filtering, sleek animations, and a responsive layout tailored for a seamless user experience.
  </p>
</div>

## ✨ Features

- **Modern UI/UX Design**: Clean, premium aesthetic featuring glassmorphism effects, dynamic glow orbs, and smooth micro-interactions.
- **Dynamic Product Selection**: Filter products by top brands (Apple, boAt, Noise, Fire-Boltt) dynamically using JavaScript.
- **Responsive Navigation & Layout**: Fully optimized for desktops, tablets, and mobile devices.
- **Interactive Elements**: Floating UI cards, an automatic rotating brand ticker, and functional cart/search button states.
- **Typography & Iconography**: Integrated with lightweight Phosphor Icons and modern Google Fonts (*Outfit* and *Space Grotesk*).

## 🛠️ Tech Stack

- **HTML5**: Semantic and accessible markup.
- **CSS3**: Custom styling, CSS Variables, Flexbox/Grid layouts, and CSS keyframe animations. No external UI frameworks were used.
- **Vanilla JavaScript**: DOM manipulation for product filtering logic, dynamic rendering, and interactive UI component states.

## 🚀 Getting Started

To view the project locally, there is no need for complex build tools or package managers.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/RohanAdkine/Smartwatches_Ecommerce.git
   ```

2. **Open the project:**
   Simply open the `index.html` file in your preferred web browser. 

## 📁 Project Structure

- `index.html` - The main layout and structure of the application.
- `style.css` - All custom styling, animations, and responsive media queries.
- `script.js` - Logic for dynamic product injection and category filtering.
- `assets/` - Folder containing graphical assets and images used in the project.
